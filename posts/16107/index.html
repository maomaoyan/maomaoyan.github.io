Markdown文档初始化

部署

新建仓库，库名格式为：username.github.io
github只能一个用户只能建一个主页。

插件

hexo-abbrlink2

基于start变量的自动递增永久链接
npm install hexo-abbrlink2 –save
修改_config.yml文件夹：
permalink: posts/:abbrlink/
abbrlink:
start: 1000 # the first id, default 0

源代码备份自动化

源文件备份

把项目中themes\butterfly.git 隐藏文件夹直接删掉
push上去就行了

自动部署

先申请一个用来Github Action的Token

申请完记得保存Token，只显示一次！
新建一个私有仓库，建议初始化下。
.github文件夹下新建workflows文件夹，再在workflows文件夹内新建autodeploy.yml

完事push上去，可以查看源代码仓库的Actions查看部署运行情况
以后更新博客就别用hexo命令，直接正常提交代码就行了。
